# Cart Optimizer — 凑单优化器
淘宝/京东/拼多多满减凑单智能计算，帮你用最少额外支出触发最大优惠。

## What it does
- 🧮 凑单计算：输入购物车金额和满减规则，自动计算最低凑单金额
- 🛒 凑单品比选：多个候选凑单品中推荐最省钱的组合方案
- 🔗 多券叠加判断：分析平台券、店铺券、品类券的叠加规则与可行性

## Example scenarios

**1. 凑单被套路了怎么办**
> 👤 淘宝618凑了300减50，结果发现凑的单品没法退，硬花了200块买了一堆没用的，下次怎么凑才不亏？
> 🤖 分析凑单反被套路的常见原因：凑单品属于'不支持单独退款'类目。推荐正确策略：优先选天猫超市日用品当凑单品（可单独退）；利用价保工具退差价；使用购物车分摊工具计算最优凑单倍数。建议付款前截图对比实际折扣率。

**2. 候选凑单品选最优**
> 👤 购物车 286，满 300 减 40，可选凑单品：纸巾 12，牙线 18，收纳盒 35。帮我选最划算方案。
> 🤖 逐一评估：加纸巾（小计298不够）、加牙线（小计304刚好触发，成本18）、加两个（小计316多花30）。推荐只加牙线，成本最低即可触发优惠。计算最终支付、节省金额和是否值得保留凑单品。

**3. 多券叠加检查**
> 👤 当前 420，平台满 500 减 80，店铺券满 399 减 30，品类券满 450 减 20，哪些能叠加？
> 🤖 分析各券状态：店铺券已触发（420>399），品类券差30（需450），平台券差80（需500）。若凑到500可同时触发三券叠加。输出平台叠加规则注意事项和建议。
